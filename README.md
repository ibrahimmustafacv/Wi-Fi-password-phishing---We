## 🧠 نظرة عامة | About
**Wi‑Fi Password Phishing — WE** هي عبارة عن **صفحة ويب تحاكي واجهة تسجيل الدخول** لشبكة “We” المصرية، مُزيّنة بعرض وهمي (هدية 300 جيجابايت مجانية) بهدف خداع المستخدم وإدخال كلمة مرور الراوتر. عند إرسال البيانات، يتم توجيهها فوراً إلى **بوت تيليجرام** خاص بك عبر واجهة برمجة التطبيقات (API)، مما يتيح لك مشاهدة البيانات في الوقت الفعلي وكأنك تراقب هجوماً حقيقياً.

هذا المشروع **ليس أداة اختراق شبكات**، بل هو **قالب موقع إلكتروني** (HTML/CSS/JS + Backend) يُمكن تشغيله إما على **الخادم المحلي** (للتجارب الشخصية) أو رفعه على **استضافة ويب** (لمحاكاة سيناريوهات الهندسة الاجتماعية عن بُعد).

**لا يجوز استخدام هذا الموقع لسرقة بيانات حقيقية أو استهداف أي شخص دون إذن صريح مسبق.**

## ✨ الميزات | Features

- **واجهة مستخدم واقعية** – نسخة طبق الأصل من صفحة We مع تحسينات للهواتف المحمولة.
- **إشعارات فورية** – إرسال كلمة المرور والبيانات المدخلة إلى بوت تيليجرام الخاص بك مباشرة.
- **خاصية التوجيه (Redirect)** – بعد الضغط على إرسال، يتم توجيه المستخدم إلى موقع حقيقي (مثل صفحة We الرسمية) لتقليل الشك.
- **دعم اللغتين العربية والإنجليزية** – سهولة التعديل حسب الجمهور المستهدف.
- **نظام تخزين اختياري** – إمكانية حفظ البيانات في ملف (log) محلي إلى جانب إرسالها للتيليجرام.
- **خفيف الوزن** – لا يحتاج إلى قواعد بيانات أو إعدادات معقدة.
- **التوافق مع الاستضافات** – يعمل على أي مستضيف يدعم PHP (أو Python/Node.js حسب الإصدار).
- **التشغيل المحلي** – يمكن اختباره بسهولة عبر XAMPP، WAMP، أو `php -S`.

## ⚙️ آلية العمل | How It Works

1. **فتح الرابط** – يصل المستخدم إلى الصفحة (سواء عبر رابط محلي `localhost` أو رابط استضافة).
2. **عرض الواجهة المخادعة** – تظهر صفحة احتجاز تحتوي على نموذج إدخال كلمة المرور مع رسالة تحفيزية (مثل: “سجل الآن لتحصل على 300 جيجا مجاناً”).
3. **إدخال البيانات** – يقوم المستخدم بإدخال رقم الهاتف وكلمة مرور الواي فاي (أو أي بيانات مطلوبة).
4. **الإرسال** – عند الضغط على زر “تسجيل الدخول”، يتم إرسال البيانات إلى النص الخلفي (Backend) عبر طلب `POST`.
5. **التنبيه الفوري** – يقوم النص الخلفي بتحويل البيانات إلى رسالة وإرسالها إلى بوت التيليجرام الخاص بك عبر واجهة برمجة التطبيقات (API).
6. **إعادة التوجيه** – في نفس الوقت، يتم إعادة توجيه المستخدم إلى موقع حقيقي (مثل الموقع الرسمي لـ We) لإيهامه بأن العملية ناجحة.

## 🖥️ متطلبات التشغيل | Requirements

لتشغيل الموقع محلياً أو على استضافة، ستحتاج إلى:

- **بيئة تشغيل ويب** (Apache, Nginx, أو خادم PHP المدمج).
- **PHP إصدار 7.4+** (إذا كان الباكند مكتوباً بـ PHP).
- **اتصال بالإنترنت** – للتواصل مع واجهة برمجة تطبيقات تيليجرام.
- **بوت تيليجرام** مع `Bot Token` و `Chat ID` الخاص بك (للاستقبال).
